使用队列来实现栈

题目表述：

使用队列实现栈的下列操作：

push(x) – 元素 x 入栈

pop() – 移除栈顶元素

top() – 获取栈顶元素

empty() – 返回栈是否为空

个人总结：

  • 首先需要了解队列和栈的特性,队列是先进先出,栈是后进先出。其实这个只是一个总体的概括,如果时间长了你可能会理解到这种特性更加深刻的特性。。。

  • 使用队列来实现栈的话,我们可以只用一个队列,对队列来进行遍历,只留下最后一个元素,这样在这个元素出队列的时候其实就是当前队列里面最有一个元素出队列,通过这种方式来实现栈的后进选出操作。

    class SolutionX:
     def __init__(self):
         from collections import deque
         self.q = deque()

     def push(self, value):
         self.q.append(value)
         n = len(self.q)
         while n > 1:
             temp = self.q.popleft()
             self.q.append(temp)
             n -= 1

     def pop(self):def top(self) -> int:
         if not self.empty():
             lenght = len(self.stack)
             while lenght > 1:
                 self.temp.append(self.stack.pop(0))
                 lenght -= 1
             temp = self.stack.pop(0)
             self.temp.append(temp)
             self.temp, self.stack = self.stack, self.temp
             return temp

     def empty(self) -> bool:
         return self.stack == []
         n = len(self.q)
         if n:
             return self.q.popleft()
         else:
             return -1
